使用GitBook

安装node

1.直接到nodejs官网下载，简单安装即可 2.安装完node -v 检查是否安装成功

安装Gitbook

1.npm install gitbook -g 安装gitbook 2.npm install -g gitbook-cli 安装gitbook-cli 3.gitbook -V 检查是否成功

如何使用GiBook

1.GitBook Editor编辑完book后保存 2.找到并进入GitBook Editor保存的book目录

cd C:/Users/ben/GitBook/Library/Import

3.运行GitBook查看book效果

gitbook serve

4.访问http://localhost:4000/ 查看相应的book
